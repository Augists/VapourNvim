# VapourNvim

Goal of the project

* Functionality
* Minimalism
* Get Work Done

## Installation

You can clone the repo and then install all the plugins

```bash
git clone https://github.com/hackorum/nvim ~/.config/nvim
# Install the plugins
nvim
:PackerSync
```
