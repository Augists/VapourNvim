# VapourNvim

Goal of the project

* Functionality
* Minimalism
* Get Work Done

## Installation

First you need to have neovim 0.5 otherwise this will not work
You can clone the repo and then install all the plugins

```bash
git clone https://github.com/hackorum/VapourNvim ~/.config/nvim
nvim
:PackerSync
```
